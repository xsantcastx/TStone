# TopStone — Logo Pack

This folder contains ready-to-use SVG assets in light/dark and two lockups.

## Files
- `topstone-mark-dark.svg` — Icon mark for light backgrounds (charcoal slab, bone carvings, bronze bevel)
- `topstone-mark-light.svg` — Icon mark for dark backgrounds (white slab, dark carvings, bronze bevel)
- `topstone-logotype-horizontal.svg` — Icon + TOPSTONE wordmark + tagline
- `topstone-logotype-stacked.svg` — Stacked layout (icon above wordmark + tagline)

## Colors
- Ink (charcoal): #0B0B0C
- Bone (light):   #F3F2EF
- Bronze (accent): #B08968

## Notes
- The logotype variants use system fonts (Cinzel/Cormorant + Inter). If you need 100% identical rendering across devices, ask for the **outlined** (paths) version and I’ll provide it.
- You can recolor by editing hex codes directly inside the SVG or using CSS `filter: invert(1)` for quick reversal of the mark.
- Recommended clearspace: height of the “T” around the wordmark; min width for full lockup on web: 120px.
